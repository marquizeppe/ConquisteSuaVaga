Minha primeira página utilizando HTML e CSS

Essa página está sendo criada no dia 23/01/2022 marcando o início do meu primeiro projeto em HTML e CSS. Utilizarei as aulas do Pedro Marins disponíveis na Alura para me guiar nesse projeto, contudo, cada tag utilizada será registrada no meu caderno de estudos no notion para que a partir desse projeto eu saiba iniciar qualquer outro projeto apenas consultando minhas anotações.

Ao longo de 4 dias saí do absoluto zero no quesito mão na massa e aprendi a montar uma estrutura básica de uma página Html. Em quatro dias de estudos e prática de Html e Css pude aprender e utilizar os seguintes conceitos:

  • !DOCTYPE html - Indicar para o navegador a versão Html que estamos utilizando
  • head - Local onde passamos as informações que serão lidas pelo navegador
  • meta charset utf8 - O conjunto de caracteres para que acentos e caracteres esoeciais sejam exibidos
  • link rel stylesheet - Criar um arquivo externo para estilizar no CSS a página Html
  • body - Informações que serão exibidas para o usuário
  • header - Cabeçalho da página
  • img - Inserir uma imagem na página
  • h1,h2,h3... - Criar títulos para uma sessão
  • p - Tag para indicar parágrafos
  • ul - Tag para criar listas não ordenadas (ol para listas ordenadas)
  • div - Tag para separar blocos de conteúdos
  • GitHub - Enviar commit de cada alteração do código para o meu repositório

Hoje, dia 31/01/2022 será dado início ao segundo projeto e no dia 07/02 você poderá acessar a segunda página Html e acompanhar a minha evolução.
